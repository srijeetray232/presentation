Event-Driven Kudos Messaging

SAP Advanced Event Mesh (Solace)

Event-driven architecture using SAP Advanced Event Mesh powered by Solace. Implements Publisher-Subscriber pattern for asynchronous kudos messaging across enterprise applications.

SAP Event Mesh Solace Publisher-Subscriber Async Messaging Mail Integration

Key Features:

  • Decoupled event-driven architecture
  • Asynchronous message processing
  • Topic-based message routing
  • Automatic email notifications

Architecture Components:

  • Publisher iFlow: Accepts kudos messages and publishes to Solace queue
  • Subscriber iFlow: Listens to queue, formats messages, sends emails
  • Topic Structure: company/employee/kudos
  • QoS: At Least Once delivery guarantee

Message Flow:

HTTP trigger → Content modifier → AEM publish → Queue → AEM subscribe → Email format → Mail delivery
